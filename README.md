📊 Customer Behavior Data Analysis

An end-to-end Data Analytics portfolio project focused on analyzing customer shopping behavior using Python, PostgreSQL, and Power BI. This project demonstrates the complete analytics workflow—from data cleaning and transformation to SQL-based business analysis and interactive dashboard creation.

📌 Overview

The objective of this project is to analyze customer purchasing behavior and generate actionable business insights that can support data-driven decision-making.

The project covers:

Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
SQL-based business analysis
Interactive Power BI dashboard
Business insights and recommendations

🚀 How to Run
Clone this repository
git clone https://github.com/your-username/customer-behavior-data-analysis.git
Open customer_analysis.ipynb in Google Colab or Jupyter Notebook.
Install the required libraries.
pip install pandas sqlalchemy psycopg2-binary
Run the notebook to clean and prepare the dataset.
Execute the SQL queries in PostgreSQL.
Open customer_dashboard.pbix in Power BI Desktop to explore the dashboard.
